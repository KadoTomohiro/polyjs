Mazes from OFF files

maze.js

A maze can be generated from a basic surface described in OFF file format as a set of points and a set of cells, with each cell having a list of points as its boundary. Each cell adjoins several others and we can form a graph with cells as nodes and walls as edges. We can then generate a maze by finding a spanning tree over this graph (here we use Kruskal's algorithm). The exact topology of the surface isn't important, as long as each wall connects at most 2 cells.
